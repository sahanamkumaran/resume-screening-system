# Resume Screening System

## Overview
An AI-powered Resume Screening System that matches candidate resumes with job descriptions using NLP and BERT embeddings.

## Technologies Used
- Python
- NLP
- BERT
- Sentence Transformers
- Streamlit
- Scikit-Learn

## Features
- Resume Matching
- Semantic Similarity Analysis
- BERT Embeddings
- Candidate Ranking
- Streamlit Dashboard

## Workflow
1. Input Job Description
2. Upload or Paste Resume
3. Generate BERT Embeddings
4. Compute Similarity Score
5. Display Match Percentage
